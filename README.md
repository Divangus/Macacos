# Teen Mutant Ninja Turtles

## Description

This game is a tribute of the original TMNT game, developed by [Konami](https://www.konami.com/en/).

## Key Features

 - 1 player
 - Control with the keyboard
 - Press Space to start the game
 
## Controls

- W, A, S, D  - Movement
- Space - Jump
- K - Attack

### Debug functionality

- F1 - Colisions
- F2 - God Mode
- F3 - Direct win/lose
