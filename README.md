# Teen Mutant Ninja Turtles
(https://github.com/Divangus/Macacos)

## Description

This game is a tribute of the original TMNT game, developed by Konami.

## Key Features

 - 1 player
 - Control with the keyboard
 
## Controls

- W, A, S, D  - Movement
- Space - Jump
- K -Attack

###Debug functionality

- F1 - Colisions
- F2 - God Mode
- F3 - Direct win/lose
